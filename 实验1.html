前言

SGML 是 HTML 的超集，标准通用标签语言。规定了独立于平台和应用程序的文本文 档的格式、索引和链接信息。描述：结构（DTD），内容（标题、段落、项目、列表）和样 式（DSSSL-文档样式语义和规范语言）。
欧洲原子核研究委员会研究用 Berners-Lee 发明了 HTML 语言。Html 的出现就是为了 避免互联网的分裂。避免“鸡同鸭讲”。
浏览器就是 html 的解释器或者翻译机。除了我们常用的浏览器，国外还特别重视弱势 群体获取信息的权利，在 html 标准制定时，有些规定是专门为了特点的语音合成设备制定 的，以帮助盲人阅读到 html 上表述的信息。那么整个语音合成器也是 html 的一个解释器和 翻译机
